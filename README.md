# MAtratum
数式を投稿できるSNSです。<br/>
[MAtratum-数式を投稿できるSNS-](http://34.168.80.237:80)

## Requirements
* Docker
* Docker Compose

## Installation
```sh
git clone -b toGCE https://github.com/Eggplanck/MAtratum.git
```

## Usage
```sh
cd MAtratum
docker compose build
docker compose up -d
```
stop
```sh
docker compose down
```
